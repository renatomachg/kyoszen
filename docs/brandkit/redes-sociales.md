## Formatos de publicacion (dimensiones)

### Instagram
- Post cuadrado: 1080 x 1080 px
- Story: 1080 x 1920 px
- Reels: 1080 x 1920 px

### Facebook
- Post: 1200 x 630 px
- Story: 1080 x 1920 px
- Cover: 851 x 315 px

### LinkedIn
- Post imagen: 1200 x 627 px
- Cover empresa: 1128 x 191 px
- Logo empresa: 300 x 300 px

## Lineamientos visuales

- Usar paleta de colores oficial (navy, blue, yellow, blanco)
- Tipografia: DM Sans
- Sin acentos en textos (convencion del cliente)
- Logo siempre visible en esquina

## Ideas de contenido

- Vacantes disponibles (recurrente)
- Tips para candidatos: como preparar documentacion, entrevistas
- Tips para empresas: como reducir rotacion, cuando tercerizar RRHH
- Datos del sector RRHH en Mexico
- Testimonios / casos de exito (cuando el cliente autorice)
- Presentacion del equipo / cultura de Kyoszen
