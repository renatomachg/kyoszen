# Logos

Carpeta para archivos de logo del cliente.

## Pendiente

- [ ] Logo PNG del cliente (fondo transparente, version principal)
- [ ] Logo PNG version blanca (para fondos oscuros)
- [ ] Logo PNG version oscura / navy (para fondos claros)
- [ ] Favicon (32x32 y 64x64)

## Uso actual

Mientras no se reciba el logo oficial, el sitio usa `KyoszenLogo.tsx` (componente SVG generado).
Al recibir el PNG, reemplazar en `src/components/layout/Navbar.tsx` y `Footer.tsx`.
